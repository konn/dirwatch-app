dirwatch -- Directory Change Notifier for OS X
==============================================

Install
-------

```sh
$ git clone https://github.com/konn/dirwatch-app.git
$ cd dirwatch-app
$ stack install
```

Usage
-----

```sh
$ dirwatch path/to/observe foo
```

You can recieve notification via Notification Center.
